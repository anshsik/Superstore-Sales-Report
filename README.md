# Superstore Sales Analysis Report 
This project presents an interactive Superstore Sales Analysis dashboard designed to analyze sales performance, order trends, product behavior, and customer insights across multiple business dimensions.

## Project Objective

The main objective of this project is to:
- Analyze sales and order patterns over time

- Understand product category and sub-category performance

- Evaluate regional and city-level sales contribution

- Study customer distribution and target achievement

- Provide high-level KPIs for quick business overview

## Key Analysis & Insights
- **Sales & Order Trends**: The dashboard captures sales and order behavior across:
  - Daily trends
  - Monthly trends
  - Yearly trends
This helps in identifying growth patterns, seasonality, and fluctuations in business performance.
  
- **Category & Product Performance**: The analysis provides insights into:
  - Sales contribution by product category
  - Order distribution across sub-categories
  - Product-level performance using tabular views including total orders and sales
This helps identify high-performing and low-performing products.

- **Geographical Analysis**: Geographical insights include:
  - Sales distribution across regions
  - City-wise contribution to total sales
  - State-level filtering for focused analysis
These insights support location-based business strategy and expansion planning.

- **Customer Analysis**: Customer-centric analysis includes:
  - Total number of customers compared against target
  - Customer-wise order and product details
  - Customer distribution across regions and countries
This helps in understanding customer reach and engagement.

## Key KPIs
The dashboard highlights key performance indicators that dynamically update with filters:
- Total Sales
- Total Orders
- Total Products
- Total Customers
- Total Number of States
These KPIs provide a quick snapshot of overall business health.
## Tools and Technologies

- **Python Libraries**: Used for data cleaning and transformation. Libraries such as Pandas and NumPy facilitated data manipulation, while Matplotlib and Seaborn were employed for visualization.

- **Power BI**: Utilized to create an interactive dashboard that showcases the key insights and visualizations. The dashboard allows for dynamic exploration of the election data, highlighting seat distribution, vote shares, and top parties.
